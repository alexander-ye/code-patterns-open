## Problem Statement

<blockquote src={frontmatter.src_url}>

Implement `pow(x, n)`, which calculates `x` raised to the power `n` (i.e., <code>x<sup>n</sup></code>).

**Constraints**

- <code>
    -100.0 {"<="} x {"<="} 100.0
  </code>
- <code>
    -2<sup>31</sup> {"<="} n {"<="} 2<sup>31</sup>-1
  </code>
- `n` is an integer.
- Either `x` is not zero or `n` > 0.
- <code>
    -10<sup>4</sup> {"<="} x<sup>n</sup> {"<="} 10<sup>4</sup>
  </code>

<details>
<summary><strong>Example Test Cases</strong></summary>

**Example 1:**

```
Input: x = 2.00000, n = 10
Output: 1024.00000
```

**Example 2:**

```
Input: x = 2.10000, n = 3
Output: 9.26100
```

**Example 3:**

```
Input: x = 2.00000, n = -2
Output: 0.25000
Explanation: 2-2 = 1/22 = 1/4 = 0.25
```

</details>
</blockquote>

## Solution

```python
class Solution:
  def myPow(self, x: float, n: int) -> float:
    return x**n
```
